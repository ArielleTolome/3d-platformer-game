# Sound Files

This directory contains the sound effects for the 3D platformer game.

## Required Sound Files:

1. **jump.mp3** - Sound effect when player jumps
2. **move.mp3** - Sound effect when player switches lanes  
3. **coin.mp3** - Sound effect when collecting coins
4. **gameOver.mp3** - Sound effect when game ends
5. **backgroundMusic.mp3** - Background music during gameplay

## Recommended Audio Formats:
- MP3 for broad browser compatibility
- OGG as fallback for Firefox
- WAV for high quality (larger file size)

## File Requirements:
- Keep files small (< 1MB each for sound effects)
- Background music can be larger but consider loading time
- Use royalty-free or original audio content only

## Where to Find Sounds:
- Freesound.org (requires attribution)
- Pixabay Audio
- OpenGameArt.org
- Create your own using tools like Audacity
